Description

Plaintxt is a minimalist text editor designed for macOS that focuses on clarity and productivity. Built with SwiftUI, it provides a clean, distraction-free environment for writing and editing plain text documents with seamless iCloud integration.

Key Features

Clean Interface

Monospace font with customizable size (9-24pt) for better readability and focus on your content.

iCloud Sync

Automatic document synchronization across all your devices using Apple's secure iCloud infrastructure.

Document Management

Native macOS document browser integration for easy file organization and management.

Word & Character Count

Real-time statistics with toggle visibility to track your writing progress.

Keyboard Shortcuts

Full keyboard navigation and shortcuts for power users and accessibility.

Accessibility

Complete VoiceOver and accessibility support with full keyboard navigation.

Technical Specifications

  • Minimum macOS Version: 13.0 (Ventura)
  • Architecture: Universal (Intel x64 + Apple Silicon)
  • File Size: ~2.5 MB
  • Memory Usage: ~15-25 MB typical
  • Storage: Documents stored in iCloud Drive (optional)
  • File Format: UTF-8 plain text (.txt) files
  • Dependencies: Pure SwiftUI implementation

  • Memory: 4 GB RAM minimum
